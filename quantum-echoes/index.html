About

Features

  • A brush tool to set initial conditions and regions of potential energy with the mouse
  • Creating particles as Gaussian wave packets with an initial momentum vector
  • Four visibility layers: Wave Function, Probability, and Real and Imaginary components
  • Three boundary condition options: Dirichlet, Neumann, and Periodic
  • Leapfrog integration as described in this paper

How It Works

This project evolves a quantum wave function over time by numerically solving the Schr√∂dinger Equation:

Since Œ® is a complex valued function, this equation can be expanded, resulting in two coupled partial differential equations (‚Ñè has been scaled to 1):

Here, u and v are the real and imaginary components of the wave function respectively. V(x,y) is the potential energy of the domain. All three of these are functions: V(x,y) is a function of space while u and v are functions of both space and time.

These functions are discretized into 2D grids and a technique called Leapfrog Integration is used to numerically approximate solutions to the above equations. All of this is done on the GPU to massively parallelize computation.

Of course, there is a lot more to this project than just that, but this is the foundation of it all.
